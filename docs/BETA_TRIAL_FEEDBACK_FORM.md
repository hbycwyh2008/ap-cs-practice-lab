## For Form Creators: Distribution Tips

**How to Use This Template:**

1. **Choose a Form Platform:**
   - Google Forms (easiest, free)
   - 腾讯问卷 (Wenjuan.com) (for China)
   - Feishu Forms (for corporate/school users)
   - Typeform (for better UX)
   - Microsoft Forms (for O365 users)

2. **Copy Questions:**
   - Use section headers to organize
   - Include all question types (scale, multiple choice, long answer)
   - Make critical questions required (★)
   - Make optional questions non-required

3. **Customize:**
   - Add school logo if appropriate
   - Change contact information
   - Adjust language for your audience
   - Remove questions not relevant to your trial

4. **Test First:**
   - Fill out the form yourself
   - Check estimated time (should be 10-15 minutes)
   - Ensure all logic branches work
   - Verify response collection

5. **Distribute:**
   - Send via email to beta teachers
   - Set deadline (e.g., within 1 week of trial end)
   - Send reminder after 3 days
   - Thank responders

6. **Analyze:**
   - Export to spreadsheet
   - Calculate averages for scale questions
   - Tag themes in open responses
   - Prioritize by frequency and severity

7. **Follow Up:**
   - Share summary with participants
   - Thank them publicly (if they agreed)
   - Update roadmap based on feedback
   - Invite top respondents to deeper discussions
