## 正文版本 1：专业版

```
我正在做一个 AP CSA Java FRQ 自动判题练习平台，目前进入小范围 Beta 试用阶段，邀请 3-5 位 AP CSA / Java 老师参与测试。

【平台功能】
✅ Java FRQ 自动判题（public/hidden tests）
✅ 老师题库管理（创建、归档、标签）
✅ 自动组卷（按技能和难度筛选）
✅ 学生提交流程（run tests → submit final）
✅ 老师 Dashboard（完成率、题目性能、技能分析）
✅ CSV 导出（用于成绩记录）
✅ 匿名学生账号（保护隐私，适合 Beta）

【当前范围】
• AP CSA only（还没做 AP CSP）
• Java Array FRQ 为主（traversal、manipulation）
• public int solve(int[] nums) 风格
• 测试驱动评分（test-based grading）
• 不包括 AI 反馈、rubric scoring

【Beta 试用细节】
• 时长：2-4 周
• 学生规模：5-20 人/老师
• 作业数量：1-2 个 Java Array 作业
• 账号方式：匿名 demo 账号（不用真实学生数据）
• 反馈方式：结构化问卷 + 可选访谈

【适合谁】
- AP CSA 老师（国际学校、公立学校、培训机构）
- Java 老师（希望学生反复练习 FRQ）
- 愿意尝试教育科技新工具的老师
- 能提供建设性反馈的老师

【不适合谁】
- 只教 AP CSP 的老师（暂不支持）
- 需要立即投入生产环境（这是 Beta）
- 期望 AI 智能批改的老师（当前只有 test-based）

【隐私保护】
• 使用匿名学生账号（student-001@class-X.demo 格式）
• 不收集真实学生个人信息
• Beta 阶段数据透明，可导出可删除
• 严格遵守教育数据隐私规范

【技术栈】
FastAPI (Python) + Next.js (React) + PostgreSQL + Docker Java Runner
开源项目，代码透明

【联系方式】
如果你是 AP CSA / Java 老师，对 Beta 试用感兴趣，欢迎：
- 私信我说明基本情况（学校类型、学生规模、试用意愿）
- 或在评论区留言，我会主动联系

Beta 名额有限（3-5 位老师），先到先得。

#APCSA #Java编程 #教育科技 #国际学校 #CS老师 #自动判题 #Beta试用 #开源项目
```

**适用场景：** 专业教育账号、技术社区、详细说明场景

## 正文版本 2：口语版

```
教 AP CSA 三年了，每次布置 Java FRQ 作业，手动批改真的累。学生也不知道自己代码对不对，要等我批完才有反馈。

所以我做了这个自动判题平台，现在准备小范围 Beta 试用，找 3-5 位 AP CSA 老师一起测试。

【主要功能】
学生写 Java 代码 → 先跑 public tests 看部分结果 → 改代码 → 提交 final answer → 系统自动判全部 test cases → 老师看 dashboard 数据（哪些学生完成了、哪些题目难）

【适合什么场景】
你想让学生练 Java Array FRQ（找最大值、遍历、求和这类），但又不想每次手动批改。系统自动跑测试用例，学生立刻知道对不对，你在后台看完成率和通过率。

【Beta 试用要求】
• 2-4 周时间
• 5-20 个学生（不用全班，试点就行）
• 布置 1-2 个 Array 作业
• 用匿名账号（不需要学生真实邮箱，保护隐私）
• 试完后填个反馈表

【当前限制】
只支持 AP CSA Java（还没做 AP CSP）
只支持简单的 Array FRQ（复杂的 OOP 还不行）
没有 AI 反馈（只有 test-based 判分）

【为什么用匿名账号】
因为是 Beta，平台还在完善。匿名账号（类似 student-001@class-1.demo）保护学生隐私，也方便老师快速上手，不用担心数据安全问题。

【感兴趣的话】
私信我或评论区留言，说一下：
- 你教什么课（AP CSA / Java）
- 大概多少学生
- 什么时候方便试用

我会发你详细的试用说明和登录方式。

Beta 名额 3-5 位老师，免费试用，主要是想听听真实反馈，把平台做得更好用。

#APCSA #Java #教学工具 #国际学校 #编程教育 #自动批改 #教育科技 #免费试用
```

## 风险管理

### 潜在负面评论及应对

**评论 1："又是卖课的"**  
**回复：** 这不是卖课，是一个开源的自动判题工具，Beta 阶段免费。代码在 GitHub 上可以看到。

**评论 2："这种平台很多了"**  
**回复：** 你说得对！不过这个平台专注 AP CSA FRQ，支持匿名账号，比较适合国际学校 Beta 试用场景。欢迎试试看有什么不同 😊

**评论 3："学生会不会作弊？"**  
**回复：** 这是个好问题。平台有 hidden tests，学生看不到所有测试用例。但和所有编程练习一样，需要老师监督和教学配合。

**评论 4："不如用 LeetCode / CodeHS"**  
**回复：** 这些都是很好的平台！我这个平台更专注 AP CSA 课程场景，支持老师自建题库、自动组卷、看学生完成率。各有优势 😊

**评论 5："隐私怎么保证？"**  
**回复：** Beta 阶段用匿名账号，不收集真实学生信息。数据可导出可删除，透明公开。详细隐私政策在平台的 Beta Notice 页面。
