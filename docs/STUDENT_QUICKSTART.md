# Student Quickstart

Use this guide during AP CSA Java FRQ practice.

## 1. Log In

1. Open the class practice site.
2. Enter the account and password your teacher gave you.
3. Click **Sign In**.

If login does not work, check that you typed the email and password exactly.

## 2. Open Your Assignment

1. Go to your dashboard.
2. Find your assignment.
3. Click the assignment to open it.
4. Open the question you want to work on.

## 3. Write Your Java Code

- Read the prompt carefully.
- Write your solution in the code editor.
- Keep the required class and method names if the starter code provides them.
- Check braces, semicolons, return types, and variable names before running.

## 4. Run Public Tests

Click **Run Public Tests** to check your code against visible test cases.

Public tests help you catch basic mistakes, but they are not all the tests. Passing public tests does not guarantee full credit.

## 5. Submit Final Answer

When you are ready, click **Submit Final Answer**.

Only submit after you have reviewed your code and tested it.

## 6. If Your Code Does Not Compile

- Read the compile error message.
- Check spelling, braces, semicolons, and method signatures.
- Fix one issue at a time.
- Run public tests again after fixing the compile error.

## 7. If the Page Crashes or Freezes

- Copy your code first if possible.
- Refresh the page.
- Log in again if needed.
- Tell your teacher if the problem happens again.
