# Content Safety and Publication Policy

This project is for AP CSA classroom practice. Reference materials are allowed for concept design, but copyrighted source text must not be copied into the public repository.

## A) What the Public Repository May Include

- platform source code
- taxonomy and tagging standards
- teacher-created sample questions
- synthetic demo questions
- import tools and import format docs
- operational documentation

## B) What the Public Repository Must Not Include

- original TestDaily PDF text
- original College Board question text
- answer key dumps
- copied lecture explanations
- copyrighted screenshots
- bulk imported proprietary question banks

## C) What a Private Classroom Database May Contain

- teacher-owned materials
- licensed materials used only for internal classroom practice
- private imported question banks that are not published to this repo

## D) Safe Workflow

Reference materials -> extract concepts -> write original teacher-created practice -> tag by knowledge point -> review -> publish only if original

## Operational Rules

- Always treat external materials as references, not copy sources.
- Keep imports private unless each question is original and safe to publish.
- Before commit, review changed files for pasted copyrighted text.
- Prefer short, synthetic prompts in docs and sample JSON.
- If unsure about source rights, keep content private and do not commit.

## Publishing Checklist

- Is the question wording original?
- Are examples and test cases teacher-created or synthetic?
- Are answer explanations written from scratch?
- Are no copyrighted screenshots included?
- Are source tags and visibility tags set correctly?
