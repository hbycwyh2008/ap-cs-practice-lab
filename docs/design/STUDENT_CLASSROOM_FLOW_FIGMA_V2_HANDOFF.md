- V2 frames are editable native Figma design layers (frames, text, cards, badges, and buttons), not screenshot-only assets.

- V2 work was created on a separate page (`V2 Polished`) to avoid modifying baseline artifacts.

## Visual Improvements (V2)

- Stronger page hierarchy with clearer headers and layout grouping.
- More polished card treatment with subtle borders, shadows, and rounded corners.
- Consistent spacing system and improved readability for student metadata.
- More explicit status/type badges (`MCQ`, `FRQ`, `Mixed`, `Upcoming`, `Overdue`).
- Stronger primary action emphasis for `Start Practice` and key flow CTAs.
- Improved state design for empty/loading/error with calmer UX language.
- Cleaner MCQ and FRQ structure with clearer content/action separation.
- Improved mobile spacing and larger touch targets for assignment and MCQ flow.

## Limitations / Remaining Review Items

- This V2 is a polished design pass, not final classroom approval.
- Additional pixel-level typography/spacing tweaks may still be requested during human review.
- Copy tone and badge semantics can be further tuned after teacher review in-class context.
