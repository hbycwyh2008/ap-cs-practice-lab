# Student First Practice Handout

Welcome to AP CSA Java FRQ practice.

## Log In

1. Open the class practice website.
2. Enter the account your teacher gave you.
3. Enter the password exactly as written.
4. Click **Sign In**.

If login fails, check your spelling first. Then ask your teacher.

## Find Your Assignment

1. After logging in, open your dashboard.
2. Find the assignment for today's practice.
3. Click the assignment.
4. Open the question.

## Write Code

- Read the prompt carefully.
- Use the starter code if it is provided.
- Keep the required class name and method name.
- Write Java code in the editor.
- Check braces, semicolons, spelling, and return values.

## Run Public Tests

Click **Run Public Tests** to check your code.

Public tests are not all tests. Passing public tests means your code passed the visible examples, but hidden tests may still catch other cases.

## Submit Final Answer

When you are ready, click **Submit Final Answer**.

Submit only after you have read your code and run public tests.

## If Your Code Does Not Compile

- Read the compile message.
- Look for missing semicolons, braces, or spelling mistakes.
- Check the method signature.
- Fix one thing at a time.
- Run public tests again.

## If Tests Fail

- Read which test failed.
- Think about the input case.
- Check your loop bounds and conditionals.
- Try a small example by hand.
- Run public tests again after making changes.

## If the Page Has an Error

- Copy your code first if possible.
- Refresh the page once.
- Log in again if needed.
- Tell your teacher if the problem continues.
